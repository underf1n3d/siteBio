# Digital Ikebana | Portfolio

Personal portfolio website built with Next.js, featuring a "Digital Ikebana" aesthetic — an interactive, botanical-inspired navigation tree.

## Features
- **Botanical Navigation**: Interactive SVG tree built with Framer Motion.
- **Bilingual**: Seamless switching between Russian and English.
- **Audio Player**: Integrated BGM player with custom "scratch" effects on seek.
- **Theming**: Dark and Light modes tailored to organic colors.
- **Optimized**: No FOUC, responsive design, smooth framer-motion animations.

## Stack
- Next.js 15
- React 19
- Tailwind CSS v4
- Framer Motion

## Local Development
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build` (outputs static export to `out/`)
